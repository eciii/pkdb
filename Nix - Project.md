**Documentation**

Unfortunately the documentation is a bit scattered across three different websites:

- https://nix.dev: Home of the official documentation for the Nix ecosystem. Maintained by the _Nix documentation team_.
	- The source Gitrepo is https://github.com/NixOS/nix.dev.
- https://nixos.org: The main website.
	- The source Gitrepo is https://github.com/NixOS/nixos-homepage.
	- The "Learn" subpage collects links to several documentation resources.
- https://wiki.nixos.org: Official NixOS Wiki.

**Fact:** Documentation resources are being migrated from `nixos.org` to `nix.dev`.

It seems that there is an effort to move all documentation resources from `nixos.org` to `nix.dev`. This effort is still ongoing and the documentation resources that still remain on `nixos.org` are:

- Two of the main manuals (the Nixpkgs and NixOS manuals at `/manual/(nixpkgs|nixos)/stable`).
- The _Nix Pills_ tutorial (`guides/nix-pills`).
- The _How Nix Works_ guide (`/guides/how-nix-works`).

**Fact:** Overview of documentation resources.

_The three main manuals_

- Nix (https://nix.dev/manual/nix/stable)
- Nixpkgs (https://nixos.org/manual/nixpkgs/stable)
- NixOS (https://nixos.org/manual/nixos/stable)

Note that the Nix manual is the only one under `nix.dev`, where it is also possible to find manuals for other Nix versions: `development`, `latest`, `rolling`, `stable` and `prev-stable`. See https://nix.dev/reference/nix-manual. 

_Guides, Tutorials and similar resources_

- How Nix Works (https://nixos.org/guides/how-nix-works)
