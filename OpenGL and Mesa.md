**OpenGL**

An API specification for rendering 2D and 3D vector graphics. It is maintained by the non-profit consortium [Khronos Group](https://www.khronos.org). The main FOSS implementation of OpenGL is the Mesa 3D Graphics Library.

**Mesa 3D Graphics Library**

Hosted by freedesktop.org.

This project provides open source implementations of [OpenGL](https://www.opengl.org/) and other related API specifications.

Website: https://mesa3d.org
Gitstore: https://gitlab.freedesktop.org/mesa
